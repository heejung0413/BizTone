# BizTone

실무자가 작성한 핵심 내용(Raw Data)을 상황에 최적화된 비즈니스 텍스트로 변환하는 도구입니다.

## 주요 기능

- **채팅 변환**: 메신저/슬랙 등 채팅에 적합한 톤으로 변환
- **메일 변환**: 비즈니스 이메일에 적합한 격식체로 변환
- **보고서 변환**: 보고 문서에 적합한 공식적인 문체로 변환

## 시작하기

```bash
# 저장소 클론
git clone https://github.com/your-username/BizTone.git
cd BizTone

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 사용 예시

**입력 (Raw Data)**
```
내일 회의 10시로 변경됨. 참석자 전원 확인 필요.
```

**출력 (채팅)**
```
안녕하세요! 내일 회의가 10시로 변경되었습니다. 참석 가능 여부 확인 부탁드려요 🙏
```

**출력 (메일)**
```
안녕하세요,

내일 예정된 회의 시간이 오전 10시로 변경되었음을 안내드립니다.
참석 가능 여부를 회신해 주시면 감사하겠습니다.

감사합니다.
```
